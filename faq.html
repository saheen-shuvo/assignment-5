IMPORTANT FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that organizes HTML or XML documents into a hierarchical tree structure. It allows developers to access, manipulate, and modify the document's elements, attributes, and content using scripting languages like JavaScript.

How do you select an object from the DOM?

To select an object from the DOM, you can use methods like `document.getElementById()` for IDs, `document.querySelector()` for CSS selectors, or `document.getElementsByClassName()` for class names. These methods allow you to access and manipulate specific elements programmatically.

What is event delegation in context of DOM, and why is it useful?

Event delegation is a technique where a single event listener is added to a parent element to manage events for its child elements. It improves performance by reducing multiple listeners and simplifies handling dynamically added elements within the DOM.

How do you manipulate an element's attributes and styles using DOM?

To manipulate an element's attributes, use methods like `setAttribute()` or `removeAttribute()`. For styles, modify the `style` property directly (e.g., `element.style.color = "blue"`) or use `classList` to add, remove, or toggle CSS classes dynamically.
